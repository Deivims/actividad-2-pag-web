Resumen de HTML

Tema 1: Conceptos de HTML

Concepto Descripción
Etiqueta table Define una tabla en HTML.
Etiqueta form Define un formulario para la entrada de datos.
Atributos de table Incluye atributos como border, cellpadding, cellspacing, etc.
Etiqueta tr Define una fila en una tabla.
Atributo tr No tiene atributos específicos, pero puede tener atributos de globales.
Etiqueta td y th td define una celda en una fila, th define una celda de encabezado.
Atributos de td y th Incluye atributos como colspan, rowspan, etc.
Tablas avanzadas Uso de thead, tfoot, tbody para estructurar mejor la tabla.
Etiqueta thead Define el grupo de filas de encabezado en una tabla.
Etiqueta tfoot Define el grupo de filas de pie de tabla.
Etiqueta tbody Define el grupo de filas de cuerpo de tabla.
Etiqueta meter Define un control de entrada para medir un valor en un rango específico.
Etiqueta progress Representa el progreso de una tarea.
Etiqueta datalist Define una lista de opciones predefinidas para un input de tipo texto.
Etiqueta summary Define un resumen para un elemento details.

Tema 2: Elementos de Estructuración de una Página Web

  • Contenedor (wrapper): Un elemento que agrupa y centraliza el contenido de la página.
  • Cabecera (header): Sección superior de la página que típicamente contiene el logotipo y el menú de navegación.
  • Contenido (content): La parte principal de la página donde se muestra la información relevante.
  • Menú (menu): Navegación de la página, generalmente ubicado en la parte superior o lateral.
  • Pie (footer): Sección inferior de la página que suele contener información de contacto, enlaces y derechos de autor.
  • Lateral (sidebar): Sección a los lados del contenido principal, que puede incluir enlaces, anuncios o widgets.
  • Etiqueta div: Elemento contenedor genérico utilizado para agrupar y aplicar estilos a secciones de una página.

Tema 3: Elementos de Estructuración Semántica

  • <header>: Define la cabecera de una sección o de una página, usualmente contiene el logotipo y la navegación.
  • <nav>: Define un conjunto de enlaces de navegación.
  • <main>: Define el contenido principal de un documento.
  • <article>: Define contenido independiente y autónomo dentro de una página.
  • <section>: Define una sección dentro de un documento.
  • <aside>: Define contenido relacionado indirectamente con el contenido principal, como una barra lateral.
  • <footer>: Define el pie de una sección o de una página, incluyendo información como derechos de autor y enlaces.
  • <figure>: Define contenido ilustrativo, como imágenes o diagramas, y su <figcaption> para una leyenda.
